#### continue 문

```
var num1 = 3;
for (var i = 0; i <= 100; i++) {
    if (i % num1 == 0) {
        continue;
    }
    document.write(i + " ");
}
```

>3의 배수를 제외하고 100까지 나열...100에서 끝남

#### break 문

```
var num2 = 3;
for (var i = 0; ; i++) {
    if (i % num2 == 0) {
        continue;
    }
    if (i >= 100)
        break; // 나를 감싸고있는 첫번째 반복문만 빠져나옴
    document.write(i + " ");
}
document.write("for문 끝");
```

>"98"에서 끝남, "100"은 "99"가 올라가 "100"이 되면서 "i >= 100"에 걸려 "break;"되어서 for문을 빠져나감

#### label 문

break;인 경우

```

for (var i = 2; i <= 5; i++) {

    for (var j = 1; j <= 5; j++) {
        if ((i * j) % 2 == 0) {
            break;
        }
        document.write(i + " * " + j + " = " + (i * j) + "<br />");
    }
    document.write("for j 끝<br />");

```

>for j 끝\
