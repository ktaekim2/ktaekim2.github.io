## method(메서드)

- 특정 기능을 수행하기 위한 코드블록
- 메서드 선언 형식

```
[리턴타입] [메서드이름]() {
    메서드 호출 시 실행할 코드블록
}
```

### 메서드에서의 리턴(return)

- 해당 메서드의 코드를 모두 실행한 후 결과 값을 호출한 곳으로 넘겨주는 것
- 리턴 유무에 따라
    - 리턴 없음: void
    - 리턴 있음: 리턴 하고자 하는 데이터 타입
        - 반드시 메서드 코드블록 마지막이 return 문장 있어야 함.
