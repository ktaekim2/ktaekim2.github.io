## like 연산자

```sql
select * from emp where ename like 'm%';
-- "m"으로 시작
select * from emp where ename like '%m';
-- "m"으로 끝남
select * from emp where ename like '%m%';
-- "m"이 포함

select * from emp where ename like '_m';
select * from emp where ename like '_m%';
-- 두번째 글자가 "m"이고 뒤에는 상관 없음.
select * from emp where ename like '__m%';
```
